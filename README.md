# PyNBTExplorer
## 运行
#### 1. 安装依赖
```bash
pip install nbtlib
# 如有tkinter请跳过
# pip install tk
pip install pillow
```
#### 2. 直接打开main.py即可
或运行
```bash
python3 main.py
```
有问题欢迎提交issues
## 编译
#### 1.克隆此项目或下载main.py和main.spec
#### 2.构建项目
```bash
# 若安装pyinstaller先安装
# pip install pyinstaller
pyinstaller main.spec
```
